
1) Open terminal
2) Navigate to current folder
3) run "./orbites.sh" in terminal

note: if it says that the file is not an executable, then run "chmod u+x orbites.sh" in terminal to give required permissions

4) Wait for it to finish.

Some predefined plots can be found in corresponding folder in 'graphs'
Full data can be found in corresponding folder in 'data'

(optional) choose what to plot by manually typing or choose from plot.txt

1 Gb of storage space required.
